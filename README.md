# FlightChecker ✈️

A flight checking application built with JavaScript, CSS, and PostgreSQL.

## Overview

FlightChecker is a web application that helps users check and manage flight information. This project demonstrates a full-stack approach to building interactive flight management tools.

## Tech Stack

- **Frontend**: JavaScript (79.2%), CSS (16.2%)
- **Backend**: PLpgSQL (3.6%)
- **Markup**: HTML (1%)
- **Database**: PostgreSQL

## Features

- Flight search and availability checking
- User-friendly interface with responsive design
- Database-backed flight information storage
- Real-time flight status updates

1. Open the application in your web browser
2. Search for flights by entering your departure and arrival cities
3. Select desired dates and number of passengers
4. Browse available flights and book your preferred option
